## CDCS, CAHSS

### Module
Introduction to Python Programming

### Academic year
2023-2024

### Notes
Foundational programming teaching for mixed-discipline cohorts.

### DRPS synthesis
#### Learning objectives
- Learn the core concepts of Python programming from first principles.
- Write small programs to solve computational and data-related problems.

#### Course content
- Variables, control flow, functions, and basic program structure.
- Introductory programming exercises and small-scale problem solving.

#### Assessment profile
- Assessed through coding-focused coursework and practical exercises.
