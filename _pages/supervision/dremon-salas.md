## Dremon Salas
### MSc in Data Science for Health and Social Care (*with S. Forbes*)

### Project
Visual analytics for machine learning models in hybrid closed-loop diabetes systems.

As new statistical and machine learning models are developed for blood glucose prediction in hybrid closed-loop systems such as OpenAPS, there is a clear need to visualise both the underlying data and model outputs.

This project developed bespoke visualisations to communicate algorithmic behaviour, compare modelling approaches, and support interpretation. It also designed user-reference and dashboard components to make outputs accessible to relevant users.
