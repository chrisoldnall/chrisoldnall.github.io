## PhD Thesis Project

### Project title
Identifying causal effect sizes in population genetics via targeted Mendelian randomisation.

### Summary
This project studies causal effect estimation in biomedicine using Mendelian randomisation (MR). It focuses on robust estimation when key assumptions are challenged in practice, particularly in the presence of pleiotropy.

### Research focus
- Causal identification and estimation under unobserved confounding
- Instrumental variable methods using genetic variants
- Methodological development for pleiotropy-robust MR
